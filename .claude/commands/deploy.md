---
description: 変更をコミット、プッシュ、デプロイする一括コマンド
---

以下の手順を実行してください：

1. まず、現在の変更状況を確認する
   - `git status` を実行して変更されたファイルを確認
   - `git diff` を実行して変更内容を確認

2. すべての変更をステージングに追加
   - `git add .` を実行

3. 日本語で適切なコミットメッセージを作成してコミット
   - 変更内容を分析して、適切な日本語のコミットメッセージを作成
   - コミットメッセージの形式：
     - 1行目: 簡潔な変更概要（50文字以内）
     - 2行目: 空行
     - 3行目以降: 詳細な変更内容（必要に応じて）
   - `git commit` でコミットを実行

4. リモートリポジトリにプッシュ
   - 現在のブランチを確認
   - `git push` を実行（初回の場合は `git push -u origin <branch-name>`）

5. Cloudflare Workersにデプロイ
   - `npm run deploy` を実行
   - デプロイ結果を確認して、URLを表示

各ステップの実行結果を報告し、エラーが発生した場合は詳細を説明してください。
